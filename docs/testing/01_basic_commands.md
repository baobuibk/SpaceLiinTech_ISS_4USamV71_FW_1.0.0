# Basic Commands (UTILS)

## Expected coverage

- Display available commands
- Clear console output
- Test CLI functionality

---

## Core commands

| Command | Arguments | Expected Response |
|---|---|---|
| help | none | list of commands |
| cls | none | screen cleared |
| echo | text | echo text |

---

## Recommended command behavior

### `help`

Display all available commands in the system.

---

### `cls`

Clear the terminal screen.

---

### `echo <text>`

Print input text back to console.

- text: any string

Example:

>echo Hello
