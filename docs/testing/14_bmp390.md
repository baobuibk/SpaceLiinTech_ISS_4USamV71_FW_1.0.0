# BMP390 Module

## Expected coverage

- Read pressure/temperature from BMP390 sensor
- Support internal and external sensor
- Control I2C switch for external sensor

---

## Core commands

| Command | Arguments | Expected Response |
|---|---|---|
| bmp390_int_read | none | sensor data |
| bmp390_ext_read | none | sensor data |
| bmp390_ext_ena | none |  |
| bmp390_ext_dis | none |  |

---

## Recommended command behavior

### `bmp390_int_read`

Read data from internal BMP390 sensor.

---

### `bmp390_ext_ena`

Enable I2C switch to connect external BMP390.

---

### `bmp390_ext_dis`

Disable I2C switch.

---


### `bmp390_ext_read`

Read data from external BMP390 sensor.

Requirement:

- External I2C must be enabled
