# BME688 Test

## Expected coverage

- Initialize the BME688 environmental sensor
- Read temperature
- Read pressure
- Read humidity
- Read all environmental data
- Verify sensor communication

## Core BME688-test commands

| Command | Arguments | Expected Response |
|---|---|---|
| `bme688_init` | none | Initializes the BME688 |
| `bme688_read_temp` | none | Reads and outputs temperature (°C) |
| `bme688_read_press` | none | Reads and outputs pressure (Pa) |
| `bme688_read_hum` | none | Reads and outputs relative humidity (%) |
| `bme688_read_all` | none | Reads and prints temperature, pressure, and humidity |

## Command Usage

### 1. Initialize the BME688 sensor

**Command**

```text
bme688_init
```

**Expected Response**

```text
BME688 Init OK
```

or

```text
BME688 Init Fail
```

### 2. Read temperature

**Command**

```text
bme688_read_temp
```

**Expected Response**

```text
[BME688] Temp : 27.35 C
```

### 3. Read pressure

**Command**

```text
bme688_read_press
```

**Expected Response**

```text
[BME688] Press: 100845.72 Pa
```

### 4. Read humidity

**Command**

```text
bme688_read_hum
```

**Expected Response**

```text
[BME688] Hum  : 53.26 %
```

### 5. Read all sensor data

**Command**

```text
bme688_read_all
```

**Expected Response**

```text
[BME688] Temp : 27.35 C
[BME688] Press: 100845.72 Pa
[BME688] Hum  : 53.26 %
```

## Recommended Execution Flow

### 1. Base run

Step 1: Initialize the sensor

```text
bme688_init
```

Step 2: Read all sensor data

```text
bme688_read_all
```

### 2. Individual Sensor Test (Optional)

Read temperature

```text
bme688_read_temp
```

Read pressure

```text
bme688_read_press
```

Read humidity

```text
bme688_read_hum
```
